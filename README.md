City-Data-Analysis

This project contains a PySpark notebook for analyzing city and building data.
It includes generating demo data. The original data used for this task can be found here: https://web.archive.org/web/20241009075101/https://www.avoindata.fi/data/en_GB/dataset/postcodes/resource/3c277957-9b25-403d-b160-b61fdb47002f

How to use
1. Open the notebook (.ipynb) in Jupyter Notebook, VS Code or any compatible environment (created in Databricks).
2. Run the cells in their original order.
